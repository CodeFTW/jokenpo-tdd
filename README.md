# Jokenpo com TDD

## Instalar

Instalar [Node.js](https://nodejs.org/pt-br/download/package-manager) 18.

> Recomendo instalar com [nvm](https://github.com/nvm-sh/nvm)

Entrar nessa pasta pelo terminal e executar:

`npm install`

## Rodar Testes
 
`npm test`

## Rodar Testes com Watch
 
`npm run test:watch`
